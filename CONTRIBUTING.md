# Contributing to aryanxpatel-plugins

Thanks for your interest in contributing!

## Reporting Bugs

1. Check existing [issues](https://github.com/AryanXPatel/aryanxpatel-plugins/issues)
2. Open a new issue using the bug report template
3. Include:
   - Plugin name and version
   - Steps to reproduce
   - Expected vs actual behavior
   - Error messages (if any)

## Requesting Features

1. Open an issue using the feature request template
2. Describe the use case
3. Explain why this would be useful

## Suggesting Plugins

Have an idea for a new plugin? Open an issue with:

- Plugin name and purpose
- Key features
- Example use cases

## Code Contributions

For contributions to specific plugins, please open issues/PRs in their respective repositories:

- [gemini-image-gen](https://github.com/AryanXPatel/gemini-image-gen)

## Code of Conduct

Be respectful and constructive. We're all here to build cool tools.
